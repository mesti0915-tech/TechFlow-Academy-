# TechFlow Academy – Website Fix & Responsive Redesign

## Project Overview

This project demonstrates a **website redesign and fix** for TechFlow Academy, an educational platform. The goal was to:

- Fix layout and spacing issues  
- Make the site fully **responsive** across desktop, tablet, and mobile  
- Improve **UI/UX** for courses, instructors, and testimonials  
- Add **interactive elements** such as smooth scrolling, buttons, and a hamburger menu  
- Enhance trust with **instructor and student photos**  
- Replace text-based social links with **icons** for a modern footer  

This project is a **portfolio-ready demonstration** of frontend skills: HTML, CSS, responsive design, and modern layout techniques.

## Key Improvements (Before → After)

| Feature | Before | After |
|---------|--------|-------|
| Layout | Fixed width, messy spacing | Responsive, balanced spacing |
| Hero Section | Basic text and button | Proper spacing, interactive button with smooth scroll |
| Courses Section | Simple text boxes | Card layout with hover effects |
| Instructors | Text only | Added instructor photos in circular cards |
| Testimonials | Text only | Added student photos for realism |
| Navigation | Horizontal links, not mobile-friendly | Desktop nav + **hamburger menu for tablet/mobile** |

| Responsiveness | Not mobile-friendly | Fully responsive (desktop/tablet/mobile) |
